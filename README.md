# ts-rn-swipe-button

The swipe button

## Installation

```sh
npm install ts-rn-swipe-button
```

## Usage

```js
import TsRnSwipeButton from "ts-rn-swipe-button";

// ...

const result = await TsRnSwipeButton.multiply(3, 7);
```
